> The command keyword is "qrcode" or "vpoc".

## Project setup

Install dependencies

```bash
$ pnpm i
```

Build extension

```bash
$ pnpm run build
```

Pack extension

```bash
$ pnpm run pack
```

Publish extension (correct publisher and access token as prerequisites)

```bash
$ pnpm run publish
```
